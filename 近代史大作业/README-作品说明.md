# 长征数字地图 - 作品说明文档

> 本文件为《长征数字地图》实践作业的说明文档备选版本。
> 由于环境限制，Word文档(.docx)需要手动生成。
>
> **生成方法（二选一）：**

## 方法1：用 Python 直接运行脚本（推荐）

打开命令提示符（cmd），运行：

```bash
cd /d E:\近代史大作业
python make_docx.py
```

如果提示"python不是命令"，请用完整路径：

```bash
"D:\Python\python.exe" "E:\近代史大作业\make_docx.py"
```

## 方法2：用 pandoc 转换 Markdown

```bash
pandoc "E:\近代史大作业\doc_content.md" -o "E:\近代史大作业\长征数字地图-作品说明文档.docx"
```
